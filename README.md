#  Language Learning Quiz App

A Python-based multilingual quiz application that helps users learn vocabulary in **Tamil, Telugu, and Kannada**. The app includes difficulty levels, pronunciation support using **gTTS**, and score tracking to enhance the learning experience.

---

## Features

* Multi-language support: **Tamil, Telugu, Kannada**
* Difficulty levels: **Easy, Medium, Hard**
* Text-to-Speech pronunciation (**gTTS**)
* Randomized quiz questions
* Score tracking
* User-friendly command-line interface

---

## Technologies Used

* **Python**
* **gTTS (Google Text-to-Speech)**
* **OS module**
* **Random module**

---

##  How It Works

1. Choose a language (Tamil / Telugu / Kannada)
2. Choose a difficulty level (Easy / Medium / Hard)
3. Translate 5 randomly selected English words
4. Optionally listen to pronunciation
5. View your final score
